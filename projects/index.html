Projects

You know, the things that I've been working on lately. Also, things that I used to work on, for the historians interested in my past.

    Mixpanel

    My current full time gig. I'm a Software Engineer at Mixpanel working on their new feature flag product. When I'm not doing that, I'm building out AI coding tools and teaching about them whenever I get the chance. It's a good way to stay sharp and share what I've learned.

  •

    Sharethrough

    I worked as an Engineering Manager of the Marketplace Dynamics and MLOps team for Sharethrough, one of the largest independent ad exchanges. Of the places I've worked at, it's probably the company whose published values most closely aligned with their actual process.

  •

    Blue Matador

    At Blue Matador, I was co-founder and CTO. Blue Matador is an automated infrastructure monitoring tool that can provide actionable alerts for your AWS, K8s, and Azure infrastructures without any configuration. We ended up running out of money after 4 years, but it was a great learning experience.

  • Lucid Software

    My first job out of college. I was a backend software engineer working on Lucidchart and Lucidpress. My major projects included rewriting the real time collaboration engine, as well as the spell checker microservice.

  • ieatfit

    A side project I built for a friend. Given some basic information about your level of fitness, ieatfit will produce custom meal plans designed to help you meet your fitness goals (the service is no longer active).

  •

    Relate

    Relate is an open source database access library built for Scala with an emphasis on performance. I helped write the core of the library and designed the documentation website.

  •

    Cumulus

    Cumulus is an AWS configuration as code manager. It allows you to use simple and intuitive JSON files to manage your AWS infrastructure. I started the project, contributed many of its modules, and designed its documentation website.
